# Gemini Workspace Rules

This project uses **Bun** as its package manager and runtime. All command executions, dependency installations, and script runs should be performed using `bun`.

## Build and Dev Commands
- **Start Dev Server**: `bun run dev`
- **Build Project**: `bun run build`
- **Run Lint**: `bun run lint`
- **Install Dependencies**: `bun add <package>`
